# Rock, Paper, Scissors ✊✋✌️

A modern, responsive web-based **Rock, Paper, Scissors** game where you challenge the computer with a clean dark-themed interface and smooth animations.

## 🌟 Features

* 🎨 **Modern Dark UI:** Stylish interface with a clean, minimal design.
* 📱 **Fully Responsive:** Optimized for desktops, laptops, tablets, and mobile devices.
* ✨ **Smooth Animations:** Interactive hover and click effects for a better user experience.
* 🏆 **Live Scoreboard:** Track both your score and the computer's score in real time.
* 💾 **Session Storage:** Scores are stored using `sessionStorage` and remain available while the browser tab is open.
* 🔄 **Auto Reset:** Scores automatically reset when the browser tab or window is closed.
* ⚡ **Vanilla JavaScript:** Built with only HTML, CSS, and JavaScript—no frameworks or libraries.

## 🚀 How to Play

1. Clone this repository.
2. Open the project folder.
3. Open `index.html` in your favorite web browser.
4. Click **✊ Rock**, **✋ Paper**, or **✌️ Scissors** to make your move.
5. The computer will randomly choose its move.
6. The winner is decided instantly, and the scoreboard updates automatically.

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, CSS Variables, Responsive Media Queries)
* JavaScript (ES6)
* Session Storage API

## 🎮 Game Rules

* ✊ Rock beats ✌️ Scissors.
* ✌️ Scissors beats ✋ Paper.
* ✋ Paper beats ✊ Rock.
* If both players choose the same option, it's a draw.

Enjoy the game and see if you can beat the computer! 🎉
