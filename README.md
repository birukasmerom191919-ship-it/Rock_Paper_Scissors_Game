# Rock, Paper, Scissors ✊✋✌️

A sleek, modern web-based game where you test your luck against the computer.

## 🌟 Features
- **Modern UI:** Clean dark mode design with interactive hover animations.
- **Smart Scoring:** Uses `sessionStorage` to keep scores active.
- **Auto-Reset:** Scores automatically reset when you close the browser tab.
- **Pure Vanilla:** Built entirely with standard HTML, CSS, and JavaScript.

## 🚀 How to Play
1. Clone this repository to your local machine.
2. Open the project folder.
3. Double-click `index.html` to open it in any web browser.
4. Click on Rock, Paper, or Scissors to make your move!

Enjoy the game and try to beat the computer's winning streak!
